# capitalize-name-python

This Python program takes a user's name and capitalizes the first letter of each word.

## How it works

1. Converts the name into a list of characters.
2. Capitalizes the first letter of each word.
3. Converts the list back into a string.
4. Prints the capitalized name.

## Example

Input:
abdelrazzag abdalla

Output:
Abdelrazzag Abdalla
